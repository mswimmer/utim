Home of the Unified Threat Intelligence Model (UTIM)

This is an OWL2 ontology that tries to unify the various threat and vulnerability data formats as a single and consistent ontology. It also provides tools for doing basic conversion to an RDF model using this ontology.

core

The core ontology that others depend on. core

platform

An ontology describing the vulnerable platform. This is influenced by CPE. platform

score

An ontology for describing vulnerability metrics. Based on CVSS2 and CVSS3. score

vulnerability

A ontology for basic elements of vulnerabilities. Based on NVD. weakness

weakness

An ontology of weaknesses. Based on CWE. weakness

attack

A synthetic ontology of the Mitre ATT&CK framework based on mnemonic.no's ACT Platform code. attack.rdf. This is still highly experimental.
