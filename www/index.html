Home of the Unified Threat Intelligence Model (UTIM)

This is an OWL2 ontology that tries to unify the various threat and vulnerability data formats as a single and consistent ontology. It also provides tools for doing basic conversion to an RDF model using this ontology.

core

The core ontology that others depend on. core-1.0.0.rdf

platform

An ontology describing the vulnerable platform. This is influenced by CPE. platform-2.3-1.0.1.rdf

score

An ontology for describing vulnerability metrics. Based on CVSS2 and CVSS3. score-3.0-1.7-0.1.rdf

uvo

A unified vulnerability ontology. uvo-1.0.0.rdf

vulnerability

A ontology for basic elements of vulnerabilities. Based on NVD.

weakness

An ontology of weaknesses. Based on CWE. weakness-3.1-v1.0.1.rdf

attack

A synthetic ontology of the Mitre ATT&CK framework based on mnemonic.no's ACT Platform code. attack.rdf. This is still highly experimental.
